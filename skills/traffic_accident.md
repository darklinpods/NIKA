# 人伤案件索赔项目

## 1. 医疗费
凭医院发票据实计算。列明各家医院发票金额加总，例如：
```
30020.92 + 5575.26 + 35 + 8153.69 + ... = 合计XX元
```

## 2. 后期治疗费
参照《司法鉴定意见书》中"后续诊疗项目"，估计金额或注明"或据实结算"：
```
后期治疗费：15,000元（或据实结算，以鉴定意见为准）
```

## 3. 住院伙食补助费
```
住院伙食补助费：50元/天 × {住院天数}天 = {金额}元
```

## 4. 营养费
营养期限参照《司法鉴定意见书》医嘱/鉴定结论：
```
营养费：50元/天 × {营养天数}天 = {金额}元
```

## 5. 误工费
- 若当事人为**农村务农人员**，使用农村年平均工资：
```
误工费：54,553元/年 ÷ 365天 × {误工天数}天 = {金额}元
```
- 若能提供**工资收入证明**，按实际工资计算：
```
误工费：{月工资}元/月 × {误工月数}个月 = {金额}元
```

## 6. 护理费
护理期限参照《司法鉴定意见书》：

**护工护理**（有雇佣护工发票的部分）：
```
护工护理费：{日单价}元/天 × {天数}天 = {金额}元
```

**居民服务业标准护理**（无专职护工部分，按标准计算）：
```
家属护理费：52,532元/年 ÷ 365天 × {天数}天 = {金额}元
```

注：两部分合并为护理费总计。

## 7. 残疾赔偿金
按城镇标准计算（如当事人有证据证明住所地/经常居住地为城镇，可主张城镇标准）：
```
残疾赔偿金：46,987元/年 × {伤残赔偿系数}% × {计算年限}年 = {金额}元
```

**伤残等级对应赔偿系数**：
- 十级伤残：10%
- 九级伤残：20%
- 八级伤残：30%
- 七级伤残：40%
- 六级伤残：50%
- 五级伤残：60%
- 四级伤残：70%
- 三级伤残：80%
- 二级伤残：90%
- 一级伤残：100%

多个伤残等级合并系数：参照《最高人民法院关于审理人身损害赔偿案件适用法律若干问题的解释》规定，采用"5%折扣法"处理，主要伤残系数 + 次要伤残系数×(1-主要伤残系数)，或直接说明为"12%赔偿系数"等鉴定结论。

**计算年限**：
- 受伤时年龄不足60周岁：20年
- 60 ~ 75周岁：(75 - 受伤时年龄)年
- 75周岁以上：5年

## 8. 精神损害抚慰金
按伤残等级酌情主张（参见上方对照表），例如：
```
精神损害抚慰金：8,000元（九级伤残）
```

## 9. 交通费
凭发票据实，或根据住院天数和就医距离酌情主张：
```
交通费：{金额}元（凭票据实）
或
交通费：{金额}元（根据住院{天数}天、就医远近酌情主张）
```

## 10. 鉴定费
凭发票：
```
鉴定费：2,280元（武汉福田爱民法医司法鉴定中心）
```

## 11. 残疾辅助器具费（如有）
凭购买发票：
```
残疾辅助器具费：{金额}元
```

## 12. 被扶养人生活费（如有）
需证明被扶养人基本情况（年龄/是否有劳动能力等）。

# AI 提取与生成指令

当需要 AI 根据本 Skill 生成诉状时，请遵循以下步骤：

1. **分析案件类型**：从证据中判断是否涉及人身伤亡（人伤类）或仅有财产损失（车损类）。
2. **提取关键事实数据**：
   - 原告、被告、车主、保险公司信息
   - 事故时间、地点、经过
   - 责任认定书编号和责任划分
   - 住院天数（各家医院）、手术情况
   - 医疗费总金额（分院列举）
   - 《司法鉴定意见书》中的：伤残等级/系数、误工期/护理期/营养期天数
   - 护工费（是否雇用护工、单价、天数）
3. **套用本 Skill 的公式计算各项金额**，严格使用上方列出的基准参数。
4. **按"诉状正文结构模板"组装完整起诉状**。
5. **生成附件"索赔清单"**，每项均展示计算公式和结果。
6. **不要自由发挥法律依据**，仅在"其他情况及法律依据"节下列出案件相关的法条。

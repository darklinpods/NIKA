本文件是劳动争议纠纷诉状生成的 Skill 规则。劳动争议须先经劳动仲裁委员会仲裁，对仲裁裁决不服方可向法院起诉。

# 核心计算规则

## 1. 经济补偿金（N）
```
经济补偿金 = 月均工资 × 工作年限N
```
- 工作年限：每满1年支付1个月工资；不满6个月按0.5个月计；满6个月不满1年按1个月计
- 月均工资：离职前12个月平均工资（含奖金、津贴）
- 上限：月均工资超过当地职工月均工资3倍的，按3倍计，年限最高12年

## 2. 赔偿金（2N）—— 违法解除劳动合同
```
赔偿金 = 经济补偿金 × 2
```
适用情形：用人单位违法解除或终止劳动合同

## 3. 未签劳动合同双倍工资
```
双倍工资差额 = 月工资 × (实际工作月数 - 1)
```
- 适用期间：用工之日起满1个月至1年内未签书面劳动合同
- 最长11个月
- 注意仲裁时效：1年

## 4. 拖欠工资
```
拖欠工资 = 月工资 × 拖欠月数
```
加付赔偿金（劳动合同法第85条）：应付金额的50%~100%

## 5. 未休年假工资
```
未休年假工资 = 日工资 × 应休未休天数 × 3
日工资 = 月工资 ÷ 21.75天
```
- 工作满1年不满10年：5天年假
- 满10年不满20年：10天年假
- 满20年以上：15天年假

## 6. 社会保险补缴
列明应补缴险种（养老/医疗/失业/工伤/生育）及期间。

# AI 提取与生成指令

1. 从证据材料中提取：劳动合同、工资流水、解除通知书、仲裁裁决书
2. 计算经济补偿金/赔偿金：确认工作年限和月均工资
3. 确认仲裁前置程序是否完成（必须有仲裁裁决书才能起诉）
4. 按"诉状正文结构模板"组装完整起诉状

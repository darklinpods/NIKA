# 利率上限规则（2021年后）

- **司法保护上限**：合同成立时一年期LPR × 4
  - 2024~2025年参考：LPR 3.45% × 4 = **13.8%/年**（以实际签约时LPR为准）
- **超出部分**：法院不予支持，但已支付的不予返还
- **利息计算方式**：
  - 有约定利率：按约定（不超上限）
  - 无约定利率：按LPR计算（自逾期之日起）

# 核心索赔项目

## 1. 借款本金
```
本金 = {借款金额}元
```
凭借条/借款合同/转账记录据实主张。

## 2. 利息（借款期间）
```
利息 = 本金 × 约定年利率 × 借款天数 ÷ 365
```
- 约定利率不超过LPR×4，否则按LPR×4计
- 若无约定，借款期间不计利息

## 3. 逾期利息（逾期后）
```
逾期利息 = 本金 × 逾期利率 × 逾期天数 ÷ 365
逾期利率 = max(约定逾期利率, LPR × 1.5)，且不超过LPR × 4
```
- 计算至实际还清之日止（起诉时暂计至起诉日）

## 4. 实现债权费用（如有担保）
- 律师费：凭合同约定及发票
- 保全费：凭法院收据

# AI 提取与生成指令

1. 从证据材料中提取：借条/借款合同、转账记录、还款记录、催款记录
2. 确认借款金额、借款日期、约定利率、到期日期
3. 计算利息：按约定利率（不超LPR×4），展示计算公式
4. 计算逾期利息：从到期次日起至起诉日，展示计算公式
5. 按"诉状正文结构模板"组装完整起诉状
